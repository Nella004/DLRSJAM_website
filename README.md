# DLRSJAM – Digital Licence Renewal System Jamaica

A modern web platform designed to simplify and digitize the vehicle licence renewal process in Jamaica.

## Pages

- **Home** (`index.html`) – Landing page with overview and call to action
- **How It Works** (`how-it-works.html`) – Step-by-step guide to the renewal process
- **About** (`about.html`) – Background and mission of the project
- **Team** (`team.html`) – Meet the team behind DLRSJAM
- **Contact** (`contact.html`) – Get in touch

## Tech Stack

- HTML5, CSS3, JavaScript
- Google Fonts (Outfit, Lora)
- Hosted via GitHub Pages

## Getting Started

Clone the repo and open `index.html` in your browser — no build step required.

```bash
git clone https://github.com/Nella004/DLRSJAM_website.git
cd DLRSJAM_website
```
